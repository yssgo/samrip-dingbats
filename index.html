Sandoll 삼립 TTF Basic 은 https://spcsamlip.co.kr/sandoll-samlip-hopang/ 에서 2022년 7월에 다운로드한 Sandoll 삼립호빵체 TTF Basic 체의 폰트 파일에서 딩뱃 문자에 14자에 유니코드가 지정되어 있지 않아서 fontforge 프로그램을 이용하여 유니코드 사용자 지정 영역에서 U+E009부터 U+E016까지를 사용했습니다. 아울러 유니코드가 지정되지 않은 FT 합자, 6자와 9자의 이체자는 각각 U+FB05, U+E021, U+E022로 임의로 정하였습니다. 다른 것을 바꾸진 않았지만 혼동을 피하기 위해 글꼴 패밀리 이름을 Sandoll 삼립호빵체 TTF Basic 에서 Sandoll 삼립 TTF Basic 으로 바꾸었습니다. 폰트파일 이름은 SDSamliphopangcheTTFBasic.ttf 를 SDSamlipTTFBasic.ttf 로 바꾸었습니다. Sandoll 삼립호빵체 TTF Basic의 라이선스는 https://spcsamlip.co.kr/sandoll-samlip-hopang/ 에서 확인할 수 있습니다. 이 페이지는 https://spcsamlip.co.kr/sandoll-samlip-hopang/ 에서 오류가 수정된 파일을 제공하면 삭제할 예정입니다.
